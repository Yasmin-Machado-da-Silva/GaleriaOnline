## 🛠️ **Tecnologias Utilizadas** 🔧
- **Linguagem**: C#  
- **Framework**: ASP.NET Core Web API  
- **Padrão**: Repository  
- **DTOs**: para envio e atualização de dados  
- **Armazenamento**: Sistema de arquivos (`wwwroot/imagens`)

# ✅ **Funcionalidades Implementadas**

- [✅] Busca por ID  
- [✅] Listagem de imagens
